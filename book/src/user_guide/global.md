# Global keys

These keys perform the same action in any mode.

| Key                | Action                               |
|--------------------|--------------------------------------|
| `q`, `Q`           | Quit                                 |
| `s`, `S`           | Save a screenshot of the current map |
| `Ctrl+o`, `Ctrl+O` | Load a map from disk                 |
| `Ctrl+s`, `Ctrl+S` | Save the current map to disk         |
| `+`                | Zoom in                              |
| `-`                | Zoom out                             |
